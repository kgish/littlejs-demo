## Installation

```bash
git clone https://github.com/kgish/littlejs-demo.git
cd littlejs-demo
yarn
```

## Run

```bash
yarn dev
```

Point your favorite browser to `http://localhost:1234` and enjoy!

## Build

```bash
yarn build
```

## Lint

```bash
yarn lint
```

## Test

```bash
yarn test
```
